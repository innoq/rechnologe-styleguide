title: Sort / Filter Tables
description: A progressively enhanced sortable and filterable table

This shows the markup and components to activate the [tabelle](https://www.npmjs.com/package/tabelle) component.
In this static mockup, it is difficult to see how the component will really behave because we don't have access to the dynamic backend.

To still get a feel for how the component will behave,
we've created a demo using the static resources generated by Aiur.

Then you can also look at the code examples to see what markup you need to generate on the server to get the behavior you expect.

You can use the styling from the [Table](/table) component to get the base styling for the table.

## Filter Element

```html
<input class="tabelle-input" name="column" type="text" aria-label="Filter Column 1">
```

## Sort Element

```html
<input class="tabelle-arrow" id="column_asc" type="radio" name="sort" value="column_asc" checked="" />
<label class="tabelle-arrow--asc" for="column_asc"><span class="sr-only">Sort Column 1 Ascending</span></label>
<input class="tabelle-arrow" id="column_desc" type="radio" name="sort" value="column_desc" />
<label class="tabelle-arrow--desc" for="column_desc"><span class="sr-only">Sort Column 1 Descending</span></label>
```

## Header Element

```html
<div class="tabelle-header">
  <span class="header truncatable" id="column_group" aria-hidden="true">Column 1</span>
  <input class="tabelle-arrow" id="column_asc" type="radio" name="sort" value="column_asc" />
  <label class="tabelle-arrow--asc" for="column_asc"><span class="sr-only">Sort Column 1 Ascending</span></label>
  <input class="tabelle-arrow" id="column_desc" type="radio" name="sort" value="column_desc" checked="" />
  <label class="tabelle-arrow--desc" for="column_desc"><span class="sr-only">Sort Column 1 Descending</span></label>
  <input class="tabelle-input" name="column" type="text" aria-label="Filter Column 1" />
</div>
```
